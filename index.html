for index, crash in dc[dc["Year"] == 2019].iterrows():
  if crash['Severity'] == 1:
    folium.Marker(location=[crash["Start_Lat"], crash["Start_Lng"]], icon=folium.Icon(color='blue')).add_to(map_osm)

  if crash['Severity'] == 2:
    folium.Marker(location=[crash["Start_Lat"], crash["Start_Lng"]], icon=folium.Icon(color='green')).add_to(map_osm)

  if crash['Severity'] == 3:
    folium.Marker(location=[crash["Start_Lat"], crash["Start_Lng"]], icon=folium.Icon(color='orange')).add_to(map_osm)

  if crash['Severity'] == 4:
    folium.Marker(location=[crash["Start_Lat"], crash["Start_Lng"]], icon=folium.Icon(color='red')).add_to(map_osm)

Here is the final map. This map shows the locations of accidents in 2019, as well as there severity. Red = most severe (severity 4), orange = severity 3, green = severity 2, and blue = severity 1 (there are no recorded severity 1 accidents in 2019).

From this map we can see that most accidents in the northern part are red (severe), while there is more variety in severity in the south.
